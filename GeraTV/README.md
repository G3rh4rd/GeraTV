# RapidTV
A fast minimalist IPTV player for Android TV using Exoplayer and Firebase Database.

![iptv-github](https://user-images.githubusercontent.com/12733113/91779476-b80c6380-ec06-11ea-8f96-0ed65791a786.gif)
